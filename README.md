---

## 📌 Описание проекта

Этот проект демонстрирует полный цикл разработки LTR-модели на классическом датасете **MovieLens 1M**: от подготовки данных и создания признаков до обучения ранжирующей модели и развертывания веб-интерфейса.

Ключевая особенность — сравнение производительности "нативного" инференса CatBoost и оптимизированного через **ONNX Runtime**, а также использование метрик ранжирования (NDCG, Hit@K) для оценки качества рекомендаций.

**Стек технологий:**
* **ML Core:** CatBoost (YetiRank / PairLogit)
* **Data Processing:** Pandas, NumPy
* **Config & Logging:** Hydra / OmegaConf, Loguru
* **Deployment:** ONNX, Gradio

## 🎯 Основные цели

1.  **Построение LTR пайплайна:** Реализация `LTRDataset` для группировки данных по `UserID` и генерации признаков.
2.  **Обучение CatBoostRanker:** Настройка функции потерь (YetiRank/QuerySoftMax) для задачи ранжирования.
3.  **Оценка качества:** Расчет бизнес-метрик **Hit@5** и технических метрик **NDCG@5**.
4.  **Оптимизация инференса:** Экспорт модели в ONNX и сравнение latency (ms) с нативным исполнением.
5.  **Интерактивное демо:** Визуализация работы рекомендательной системы через Gradio UI.

## 🚀 Запуск

### 1. Подготовка данных

Скачайте MovieLens 1M и распакуйте в папку `data/`.

### 2. Обучение модели

Запуск пайплайна обучения (настройки берутся из `configs/base.yaml`):

```bash
python -m src.train

```

*Скрипт автоматически сохранит модель в `models/ranker.cbm` и выведет метрики.*

## 📊 Метрики и Результаты

В ходе экспериментов оценивались качество ранжирования и скорость ответа.

| Метрика | Значение (Test) | Комментарий |
| --- | --- | --- |
| **Hit@5** | ~0.XX | Вероятность нахождения релевантного фильма в топ-5 |

**Сравнение скорости инференса (Latency):**

| Runtime | Latency (ms/batch) | Speedup |
| --- | --- | --- |
| **CatBoost Native** | 15.2 ms | 1.0x |
| **ONNX Runtime** | **3.4 ms** | **~4.5x** |

*> Замеры произведены на CPU (Intel Core i7 / AMD Ryzen 7).*
