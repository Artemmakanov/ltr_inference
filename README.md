
# 🎬 Learn-2-rank MovieLens
**Учебный проект для изучения Learning-to-Rank (LTR) моделей в рекомендательных системах и оптимизации их инференса.**

---

## 📌 Описание проекта

Этот проект демонстрирует полный цикл разработки LTR-модели на классическом датасете **MovieLens 1M**: от подготовки данных и создания признаков до обучения ранжирующей модели и развертывания веб-интерфейса.

**Стек технологий:**
* **ML Core:** CatBoost (YetiRank / PairLogit)
* **Data Processing:** Pandas, NumPy
* **Config & Logging:** Hydra / OmegaConf, Loguru
* **Deployment:** Gradio

## 🎯 Основные цели

1.  **Построение LTR пайплайна:** Реализация `LTRDataset` для группировки данных по `UserID` и генерации признаков.
2.  **Обучение CatBoostRanker:** Настройка функции потерь (YetiRank) для задачи ранжирования.
3.  **Оценка качества:** Расчет бизнес-метрик **Hit@5** и технических метрик **NDCG@5**.
4.  **Оптимизация инференса:** Экспорт модели и сравнение latency (ms) с нативным исполнением.
5.  **Интерактивное демо:** Визуализация работы рекомендательной системы через Gradio UI.

## 🚀 Запуск

### 1. Подготовка данных

Скачайте MovieLens 1M и распакуйте в папку `data/`.

### 2. Обучение модели

Запуск пайплайна обучения (настройки берутся из `configs/base.yaml`):

```bash
python -m src.train

```

*Скрипт автоматически сохранит модель в `models/ranker.cbm` и выведет метрики.*
